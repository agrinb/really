== README

Reeally is a breakable toy that is a work in progress. It's a platform for "For Sale By Owner" 
home sellers to find and connect with RE agents on demand. 

Users can: 

* Create/edit properties

* Create/edit properties

* Register as Agents

* Agents can view appointments within their Geo-targetted radius

* Claim appointments

* Receive visitor details when appointment is claimed.

Live version is at http://reeally.heroku.com
